6. Toolbar

Most of the toolbar buttons only provide mouse-oriented shortcuts for standard menu-commands.

File → Open ...

File → Save

Reload the current file from disk. Current annotation mode and stylesheet are preserved.

Note

Note for macro developers: during this action, neither annotation mode guessing, nor file_opened_hook nor file_resumed_hook are called, but file_reloaded_hook is called instead (in case of success).

File → File Lists → Previous in File List

File → File Lists → Next in File List

File → Print ...

Undo last operation on the current tree. (See also MaxUndo configuration option.)

Redo last undone operation on the current tree.

Node → Find ...

Node → Find Next

Node → Find Previous

Tree → Previous Tree

Tree → Next Tree

View → Split View Horizontally

View → Split View Vertically

View → Close Current View

This button has two states. If pressed, all displayed views are tied together, so that pressing or buttons as well as invoking Tree → Previous Tree or Tree → Next Tree applies to all displayed views.
