Description

TrEd is a fully customizable and programmable graphical editor and viewer for tree-like structures. Among other projects, it was used as the main annotation tool for syntactical and tectogrammatical annotations in The Prague Dependency Treebank, as well as for decision-tree based morphological annotation of The Prague Arabic Dependency Treebank.

TrEd runs on the following platforms (see installation instructions for details):

  • Windows 95/98/ME or Windows NT/2000/XP (TM)
  • Linux
  • BSD, OS X, UNIX, Solaris (TM) and other UNIX-based systems

Download

Current version:

Windows
tred_wininst_en.zip
This is an installation/upgrade package intended for Windows users. It contains ActivePerl, PerlTk, and TrEd with documentation. To install or upgrade, unzip the package and run setup.bat.

WARNING

Do not use the latest ActivePerl builds 818 and 819, unless you can handle installing PPM packages by hand. The new version PPM installer introduced with these new builds seems to be completely broken on some setups. For that reason we didn't even attempt to update TrEd's installation script to be compatibe with the new PPM syntax. (Sorry, but if I can't run it, I can't test it!)

Instead, use the version ActivePerl shipped with our package above, or use build 816 from the Older Versions section of the ActiveState website.

This package is not intended for use with Perl provided by CygWin. If you realy wish to use CygWin Perl with TrEd, use Unix instalation below - but then you will be forced to use TrEd with CygWin XFree86. If you want a native Windows version, install ActivePerl (setup.bat from this package will do that for you).
Unix, Linux, Mac OS X
tred-current.tar.gz
This is the latest TrEd distribution package (including documentation). It runs on all supported platforms, but is mainly intended for Linux/Unix users. It requires Perl 5.8.3 or newer and some Perl modules provided in a dependency package below. After installing all dependencies, simply ungzip+untar this package and run tred/tred.
tred-dep-unix.tar.gz
This package bundles sources for Linux/Unix dependencies for TrEd (mostly Perl modules). The install script contained in the package automates compilation and installation of all these dependencies.
