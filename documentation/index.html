Introduction

Installation instructions | Menu commands | Viewing and editing the tree structure and attributes | How to ...

TrEd is a graphical editor for tree structures. It works with files in so called Fs-format developed by M. Kren (the same file format is used also in editor Graph) used for syntactical and tectogramatical trees in The Prague Dependency Treebank. Since Fs-format is a bit more general, TrEd may also find some other applications.

TrEd may be extended by user-defined macros written in Perl with many pre-defined functions for handling Fs-trees. Macros may be assigned keyboard shortcuts or may be inserted to user-defined menu.

There is also an non-graphical (off-line) variant of TrEd called btred - "Batch-mode Tree Editor" - which allows user to run macros on trees without having to enter the graphical interface.

TrEd runs on the following platforms (see installation instructions for details):

  • Microsoft Windows (TM) 9x,ME,NT,2000
  • Linux (with libc.6)
  • Other UNIX/Solaris platforms

Installation instructions

Introduction | Menu commands | Viewing and editing the tree structure and attributes | How to ...

On all supported platforms TrEd requires Perl to be installed and Tk toolkit (a library of GUI functions). Installation of these tools is different on all supported platforms. Installation of TrEd itself is platform-independent.

To install Perl/Tk on your system follow instructions for your platform below:

Windows

  1. Start with installing Perl support to Windows from ActiveState. This is done by running the installation program APi522e.exe from the Tools\Tree_Editors\Tred\PerlTk directory on this CD. Follow the instructions of installation program. (When the installation is done, you may need to restart Windows, so that all the changes take effect.)
  2. In this step you add Tk toolkit to Perl. If you have on-line connection to the Internet you may install the latest Tk toolkit from ActiveState's site in the following steps:
    1. Open a command prompt (MS-DOS prompt in Windows 9x/ME)
    2. Type ppm (press Enter)
    3. Type install Tk (press Enter)
    4. When the installation of Tk toolkit is finished, type exit (press Enter)
    5. Close the command prompt
    If you do not have on-line connection to the Internet, follow these steps:
    1. Open a command prompt (MS-DOS prompt in Windows 9x/ME)
    2. Change directory to Tools\Tree_Editors\Tred\PerlTk on the CD, i.e. if your CD-ROM drive is D:, type
      cd D:\Tools\Tree_Editors\Tred\PerlTk
      Under Windows 9x you may rather need to use command
      cd D:\Tools\Tree_~1\Tred\PerlTk
    3. Type ppm install Tk.ppd (press Enter)
    4. Close the command prompt when the installation of Tk toolkit is finished.
  3. In this step you install TrEd itself. It is done by copying the whole Tred folder on the CD (Tools\Tree_Editors\Tred\Tred) somewhere on your hard-disk, i.e. to folder C:\Program Files. To run TrEd double-click the tred.pl icon in the Tred folder on your hard-disk. (You may also want to create a short-cut on your desktop).

Linux

This instrutions apply only to RedHat Linux or derived distributions. If you use other distibution, follow the general UNIX instructions below.

RedHat Linux contains Perl in default installation, so we may expect that you have already Perl installed. You only have to install PerlTk toolkit and TrEd:

  1. To install PerlTk, as a root issue the following command from the Tools/Tree_Editors/Tred/PerlTk directory on the CD:
    root# rpm -i perl-Tk-800.018-1.i386.rpm
